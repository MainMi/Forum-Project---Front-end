# Node.js Lessons Forum (Frontend)

## Installation
1. **Clone the Repository:**
   ```
   git clone https://github.com/MainMi/Forum-Project---Front-end
   ```

2. **Navigate to the Project Directory:**
   ```
   cd Forum-Project---Front-end
   ```
3. **Install Dependencies:**
   ```
   npm install
   ```
4. **Run:**
   ```
   npm start dev
   ```
